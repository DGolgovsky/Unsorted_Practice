### Test project for SRT company
by Dmitry Golgovsky

##### Dependencies:
Distributive packages: `g++>=8 [clang>=6] (-std=c++17), cmake>=3.5`

##### Build:
```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
```

##### Usage:
After successful building start it with next commands:
```bash
$ cd build
$ ./parser_tests
```
